# Interactive Digestive System Simulation - Requirements

## System Requirements

### Minimum Requirements
- Browser: Modern HTML5 compatible browser (Chrome, Firefox, Safari, Edge)
- Internet: Not required (works offline)
- Storage: ~500KB for all files
- RAM: 512MB minimum
- Screen: Any resolution (responsive design)

### Recommended Requirements
- Browser: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- Internet: Initial download only (optional)
- Storage: 1MB available space
- RAM: 1GB+ for smooth experience
- Screen: Minimum 320px width for mobile

## Installation Methods

### Method 1: Direct Access (Easiest)
1. No installation needed
2. Just open index.html in browser
3. Works immediately

### Method 2: Local Server
```bash
# Python 3.x
python -m http.server 8000

# Python 2.x
python -m SimpleHTTPServer 8000

# Node.js (install first if needed)
npm install -g http-server
http-server
```
